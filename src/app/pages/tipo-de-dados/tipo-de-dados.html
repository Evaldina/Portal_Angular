Tipos de Dados

Os tipos de dados são utilizados para armazenar diferentes tipos de informação numa aplicação Angular.

Principais tipos

  • String - texto
  • Number - números
  • Boolean - verdadeiro ou falso
  • Array - coleção de valores
  • Object - conjunto de propriedades

Exemplo

let nome = "Evaldina";
let idade = 25;
let estudante = true;

Dica

Os tipos de dados ajudam a armazenar informações numa aplicação Angular.
