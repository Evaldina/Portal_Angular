Estrutura de Controles

As estruturas de controlo permitem executar diferentes ações conforme determinadas condições.

Estrutura if

let idade = 18;

if (idade >= 18) {
    console.log("Maior de idade");
}

Estrutura if...else

if (idade >= 18) {
    console.log("Maior de idade");
} else {
    console.log("Menor de idade");
}

Dica

As estruturas condicionais ajudam a tomar decisões numa aplicação.
