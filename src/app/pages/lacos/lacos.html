Laços

Os laços permitem repetir instruções várias vezes de forma automática.

Estrutura for

for(let i = 1; i <= 5; i++) {
    console.log(i);
}

Estrutura while

let contador = 1;

while(contador <= 5) {
    console.log(contador);
    contador++;
}

Dica

Utilize laços quando precisar repetir tarefas sem escrever o mesmo código várias vezes.
